# CodeAlpha Bus Pass System

This project is a Cloud-Based Bus Pass System developed as part of the CodeAlpha Cloud Computing Internship.

## Features

* Online bus pass generation
* User-friendly web interface
* Automatic ticket price calculation
* Digital bus pass generation
* Database storage using SQLite

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite

## Project Objective

The objective of this project is to create a cloud-based bus pass management system that allows users to generate digital bus passes efficiently through a web application.

## How to Run the Project

1. Install Python
2. Install Flask using:

pip install flask

3. Run the application using:

python app.py

4. Open browser and visit:

http://127.0.0.1:5000
